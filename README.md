# DeepL translate API wrapper

## Overview

DeepL translate API


## Environment values

- `AUTHKEY` : DeepL AuthKey(required)

- `BASE_URL` : DeepL API Base URL(default: `https://api-free.deepl.com`)

- `CORS` : URL if CORS would be enabled(default: ``)

- `PORT` : Port web application(default: `8080`)
